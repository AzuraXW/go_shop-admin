# 瓜喔商城后台管理系统

这是一个商城的后台管理系统，基于[vue-element-admin](https://panjiachen.gitee.io/vue-element-admin-site/zh/)开发

Api接口：[shopApi](https://github.com/AzuraXW/shopApi)

####  已实现的功能

- [x] 登录
- [x] 后台用户管理
- [x] 角色分配
- [x] 后台菜单管理
  - [x] 菜单角色分配
- [x] 角色管理
  - [x] 新建角色
  - [x] 删除角色
  - [x] 角色权限分配
- [x] 动态路由
- [x] 权限控制
